# Tentang Aplikasi

Aplikasi Antrian Bank ini merupakan project tugas akhir untuk mata kuliah Dasar Pemrograman dan Struktur Data. Program ini dibuat untuk mensimulasikan sistem antrian bank menggunakan konsep queue sebagai struktur data utama. Setiap nasabah akan mendapatkan nomor antrian otomatis berdasarkan jenis layanan, yaitu Reguler, Prioritas, dan Customer Service.

Melalui proyek ini, implementasi konsep FIFO, pengelolaan data dinamis, serta proses penanganan antrian dapat dipelajari dan diuji secara langsung. Aplikasi ini dikembangkan agar sederhana, mudah dipahami, dan dapat menjadi dasar untuk pengembangan sistem antrian yang lebih kompleks di masa depan.

# Fitur-Fitur
Aplikasi Antrian Bank ini memiliki beberapa fitur utama yang mendukung proses pengelolaan antrian, yaitu:

1. Tambah Data Antrian
Menambahkan nasabah baru ke dalam antrian sesuai jenis layanan: Reguler (R), Prioritas (P), atau Customer Service (C). Sistem otomatis menghasilkan nomor antrian seperti R001, P001, atau C001.

2. Layani Antrian
Memproses nasabah yang berada di urutan terdepan antrian. Saat dilayani, data nasabah otomatis dihapus dari queue.

3. Cari Data (ID)
Mencari nasabah dalam antrian berdasarkan ID untuk mengecek apakah data masih terdaftar serta mengetahui posisi atau informasi terkait.

4. Tampilkan Antrian Aktif
Menampilkan seluruh nasabah yang masih berada dalam antrian, lengkap dengan nomor antrian dan jenis layanan.

5. Laporan Harian
Mencatat dan menampilkan rekap data nasabah yang telah dilayani pada hari tersebut, berguna untuk dokumentasi dan evaluasi.

6. Keluar
Menutup aplikasi dengan aman setelah proses antrian selesai.
